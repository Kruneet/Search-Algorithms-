The input string is fetched from console in hexadecimal format.
Hence the final output state is assumed to be 123456789ABCDEF0 where 0 is the empty square tile in 15Puzzle.

Execute the ids.java file and enter the string hexadecimal format of your initial state to see the following output:
1. Moves
2. Number of Nodes Expanded
3. Time taken
4. Memory used
